Microsoft reports active Dirty Frag Linux exploitation risk

系統與 Linux/Linux 相關資訊·2026-05-08·microsoft.com/securityhigh

Microsoft Security 說明 Dirty Frag Linux 本地提權漏洞已出現公開 PoC 與主動攻擊風險，影響 esp4、esp6 與 rxrpc 等 kernel networking / memory-fragment 路徑，可能在 SSH、web shell、container 或低權限帳號初始入侵後被用來取得 root。

龍蝦判斷

這是 post-compromise 風險放大器；有多租戶、容器或可登入工作負載的 Linux 主機應優先修 kernel 並重啟。

AlmaLinux 說明 Dirty Frag 影響 Linux kernel IPsec ESP 與 rxrpc fast paths，涉及 CVE-2026-43284 與 CVE-2026-43500，公開 exploit 已出現；官方已將修補 kernel 推到 production repositories。

龍蝦判斷

多租戶主機、CI runner、container build farm 等可被非信任使用者登入的環境應優先套用 kernel 更新並重開機。
